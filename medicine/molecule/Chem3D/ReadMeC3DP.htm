What is the Chem3D ActiveX/Plugin?

The Chem3D ActiveX/Plugin presents interactive 3D molecular models in your web browser.  All of the visualization features of Chem3D are built into the control so that you can view and interact with molecular models.

How to use the control:
  • Use the toolbar buttons to stop the model from spinning and to change modes
  • Drag with the left mouse button for view rotation, translation, and zoom
  • Right click for additional options in the context menu
