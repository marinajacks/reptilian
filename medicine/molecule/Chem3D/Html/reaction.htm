With your creativity, you can easily develop many useful programs with the ActiveX/Plugin control.  Here is an example that shows a reaction simulation!

This animation is not a movie.
It is driven by programming on the right.
 
var ax3d = c3d_getObject("ax3d");
if (!ax3d.IsProgrammable)
 
{
	alert("You must use the Pro version of the control for this example.");
	return;
}

var bond = ax3d.Bonds.Item(0);

++n;
if (n > 10)
{
	n = 0;
	d = -d;
}

bond.Atom1.XYZ.X -= d;
bond.Atom2.XYZ.X += d;

var f = false;
f = d > 0 && n > 5 || d < 0 && n <= 5;
bond.Display.Visible = (f ? 0 : 1);

Tips:
  • You can turn off the toolbar, rotation bar and context menus by using the following script:
            toolbar=false rotationbar=false contextmenu=false
  • Instead of using a file, the input data can come from a string:
            data:[mimetype],[chemical data]
    For example: src='data:chemical/x-smiles,ClCl' 
  • There are two versions of Chem3D ActiveX/Plugin: Pro and Net.  Net is a limited version, which you can use to view chemical model.  The Pro version includes more functionality, such as the ability to access the model programmatically.
