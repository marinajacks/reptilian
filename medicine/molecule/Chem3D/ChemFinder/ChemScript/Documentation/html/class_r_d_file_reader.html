static RDFileReader* RDFileReader::OpenFile ( const std::string &  filename) [static] 
Entry of creating RDFileReader.  Usage:
Python Example:
>>> rd = RDFileReader.OpenFile('c:/my.sdf')
>>> if rd = None: pass
>>> rd.ResetReading()
>>> m = rd.ReadNext()
>>> while not rd.Eof():
...     m = rd.ReadNext()
...     if m == None:
...         print("Jump to the next structure.")
...         continue
...     print(m.Formula())
...     items = m.GetDataItems()
...     for item in items.keys():
...         print(item, '->', m[item])
...     m = rd.ReadNext()
