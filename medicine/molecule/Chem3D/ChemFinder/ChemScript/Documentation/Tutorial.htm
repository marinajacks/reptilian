1. Creating a StructureData object

(1) LoadData/LoadFile/ReadData/ReadFile

>>> m = StructureData.LoadData('C=CCO')
Open molecule successfully: chemical/x-smiles
>>> m.List()
<ChemScript14.StructureData: C3H6O ()>
atoms:
C(1)
C(2)
C(3)
O(4)
bonds:
C(1)=C(2)
C(2)-C(3)
C(3)-O(4)
>>> m = StructureData.LoadFile('c:/file01.cdxml')
Open molecule successfully: text/xml
>>> m = StructureData()
>>> m.ReadData('C=CCO')
'chemical/x-smiles'
>>> m = StructureData()
>>> m.ReadFile('c:/file01.cdxml')
'text/xml'

(2) use mime types

>>> m = StructureData.LoadData('C=CCO', 'smiles')
Open molecule successfully: chemical/x-smiles
>>> m = StructureData.LoadData('C=CCO', 'chemical/smiles')
Open molecule successfully: chemical/x-smiles
>>> m = StructureData.LoadData('C=CCO', 'cdx')
Cannot open data/file as format: chemical/x-cdx

7. Finding the largest common substructure

>>> m1 = StructureData.LoadData("C1(C)CCCC1CCO")
Open molecule successfully: chemical/x-smiles
>>> m2 = StructureData.LoadData("C1CCCC1C")
Open molecule successfully: chemical/x-smiles
>>> common = LargestCommonSubstructure.Compute(m1, m2)
>>> atommap1 = common.AtomMapM1()
>>> bondmap1 = common.BondMapM1()
>>> atommap2 = common.AtomMapM2()
>>> bondmap2 = common.BondMapM2()
>>> for a in atommap1.keys():
... print (a.Name, '->', atommap1[a].Name, '->', atommap2[a].Name)
...

C(3) -> C(3) -> C(4)

...
